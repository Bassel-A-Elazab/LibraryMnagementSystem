# LibraryMnagementSystem
This is a library management system made by core java , the following technologies were used :
- Jave.
- MySql.
# Prerequisites
- Java JDK.
- NetBeans Editor.
- MySql.

2. Admin Interface : 
   - For manage all function in library.
